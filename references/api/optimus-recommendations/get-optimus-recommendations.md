# Optimus Recommendations

- METHOD: GET
- API Endpoint URL: https://uat6www.academy.com/api/productinfo/recommendations?partNumber=<sku-id>
where,
   sku-id = SKU ID of the product for which the recommendation component is being built
- Example API Endpoint URL with SKU ID: https://uat6www.academy.com/api/productinfo/recommendations?partNumber=137180444
- Example Response: @references/api/optimus-recommendations/get-optimus-recommendations-example-response.json
